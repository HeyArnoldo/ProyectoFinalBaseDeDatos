\chapter{Lista de verificación para la entrega}
\label{app:entrega}

\begin{itemize}[label=$\square$]
  \item Se reemplazaron todos los datos entre corchetes de la portada.
  \item El resumen coincide con la versión realmente implementada.
  \item Cada requisito mantiene una prueba y una evidencia trazable.
  \item Cada evidencia de la matriz incluye su artefacto ejecutado y análisis.
  \item Las figuras tienen número, título, referencia en el texto y fuente cuando corresponde.
  \item Las tablas de resultados incluyen unidades, entorno y número de repeticiones.
  \item Las capturas no contienen datos personales, secretos ni información irrelevante.
  \item Todas las fuentes citadas aparecen en la bibliografía y viceversa.
  \item El PDF compila sin referencias, citas o marcadores sin resolver.
  \item Otra persona puede ejecutar la demostración siguiendo instrucciones escritas.
  \item Cada integrante puede explicar las decisiones, limitaciones y resultados sin depender de otro integrante.
\end{itemize}
